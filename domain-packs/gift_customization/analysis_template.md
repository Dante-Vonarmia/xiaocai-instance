# 礼品定制采购分析模板（P0）

## 1. 项目理解和核心需求
- 采购背景
- 礼品对象分层（员工/客户/VIP）
- 预算、数量、交期约束

## 2. 市场现状和分析
- 礼品定制供应市场情况
- 浅度/深度定制可得性
- 区域供给与交付能力

## 3. 成本结构分析
- 单件成本构成（材料/设计/包装/物流）
- 预算匹配与数量弹性
- 成本敏感因素

## 4. 项目风险分析
- 打样与量产一致性风险
- 交期风险
- 质量与合规风险
- 包装与运输风险

## 5. 采购策略分析
- 按定制深度的采购路径建议
- 分批采购与分层采购建议
- 供应商组合策略建议

## 6. 供应商选择建议
- longlist/shortlist 策略
- 核心筛选维度与评分逻辑
- 推荐与备选说明

## 7. 项目实施计划与执行建议
- 打样-确认-量产里程碑
- 分发执行安排
- 风险应对与兜底方案
