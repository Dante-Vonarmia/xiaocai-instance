# 活动采购分析模板（P0）

## 1. 项目理解和核心需求
- 项目背景
- 采购目标
- 核心交付范围
- 当前约束（预算、交期、地点）

## 2. 市场现状和分析
- 活动供应市场现状
- 区域资源可得性
- 供应商能力分布

## 3. 成本结构分析
- 主要成本分项（搭建、设备、执行、技术、服务）
- 预算匹配度判断
- 成本敏感项

## 4. 项目风险分析
- 交付风险
- 质量风险
- 安全与合规风险
- 供应商集中度风险

## 5. 采购策略分析
- 采购路径建议
- 分包/整包建议
- 比选策略建议

## 6. 供应商选择建议
- longlist 范围
- shortlist 推荐逻辑
- 推荐供应商能力画像

## 7. 项目实施计划与执行建议
- 里程碑计划
- 协同机制建议
- 应急预案建议
