# 礼品定制 Domain Pack（P0）

## 适用范围
- 企业礼品采购与定制（员工、客户、VIP 分层）

## 业务边界
- 负责：定制深度识别、需求字段收敛、寻源规则、风险识别、产物输出
- 不负责：品牌创意策略主导、仓储发运系统实施

## 浅度定制 vs 深度定制
- 浅度定制：标准货品 + 轻量品牌元素（Logo、简单包装）
- 深度定制：涉及产品结构/材质/模具/复杂包装/专属设计打样

## 字段分层

## 必填字段
- 项目名称
- 采购目的
- 预算金额
- 定制深度（浅度/深度）
- 数量
- 交期
- 设计要求
- 包装要求
- 分层对象（员工/客户/VIP）

## 推荐字段
- 单件目标成本区间
- 目标品质等级
- 是否需要打样与打样轮次
- 交付地区与分批要求
- 合规/环保要求

## 可选字段
- 历史礼品复盘
- 指定材质或工艺偏好
- 特定品牌限制

## 核心决策因子（必须显式评估）
- 预算
- 定制深度
- 数量
- 交期
- 设计要求
- 包装要求
- 分层对象（员工/客户/VIP）

## 供应商评估关键维度
- 定制能力（打样、工艺、设计协同）
- 产能与交付稳定性
- 质量控制与质保
- 包装与物流协同能力
- 合规资质与风险记录
- 商务条款与响应速度
- 同类案例经验

## 分析模板结构
1. 项目目标与对象分层
2. 定制复杂度评估
3. 成本与数量匹配分析
4. 交付与产能风险分析
5. 供应商策略建议
6. RFX动作建议
7. 执行计划与里程碑建议

## 寻源规则与 red flags

### 寻源规则
- 先按定制深度过滤供应商能力
- 再按交期与产能匹配过滤
- 再按质量与合规门槛筛选

### red flags
- 无同类定制案例但承诺极短交期
- 报价显著偏低且关键条款模糊
- 资质/合规材料不完整
- 打样能力与量产能力断层

## 应输出的 artifact
- 礼品结构化需求单
- 礼品需求分析文档
- 供应商筛选清单（礼品版）
- 供应商评分表（礼品版）
- shortlist 推荐与风险提示
- RFX建议单
- 澄清问题单

## 输入 / 输出 / 依赖 / 暂不做

### 输入
- 礼品定制需求字段与约束

### 输出
- 礼品场景可执行采购建议与标准产物

### 依赖
- supplier-evaluation-framework
- rfx-routing-spec
- artifact-output-spec

### 暂不做
- 礼品库存/发运/仓配系统能力
