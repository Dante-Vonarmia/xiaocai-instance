# Xiaocai Domain Standards（可交付基线）

## 使用原则

1. 本目录是 `xiaocai` 采购领域唯一可交付标准文档源。
2. `docs/discussions` 与 `docs/archive` 仅作背景参考，不作为开发验收依据。
3. 所有开发任务必须映射到本目录某一条标准条款。
